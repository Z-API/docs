---
title: "Atualizar descrição do perfil"
api: "PUT /profile-description"
description: "Altera a descrição do perfil do WhatsApp conectado à instância"
---

## Conceituação

Este método é responsável por alterar a sua descrição de perfil no WhatsApp.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Body

<ParamField body="value" type="string" required>
  Descrição do perfil
</ParamField>

---

## Request Body

```json
{
  "value": "Descrição do perfil"
}
```

---

## Response

### 200

<ResponseField name="value" type="boolean">
  `true` caso tenha dado certo e `false` em caso de falha
</ResponseField>

```json
{
  "value": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou PUT conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
