## Conceituação

Esse método ativa a leitura automática de todas as publicações de status recebidas pela API.

<Warning>
Para que funcione você deve ter a **Leitura automática** habilitada.
</Warning>

![Leitura automática de status no painel](/images/auto-read-status.jpeg)

## Atributos

### Header

<InstanceId />
<Token />
### Body

<ParamField body="value" type="boolean" required>
  `true` para ativar a leitura automática de status ou `false` para desativar
</ParamField>
