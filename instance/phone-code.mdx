## Conceituação

Permite a conexão sem a necessidade de escanear o QRCode, utilizando o recurso "Conectar com número de telefone".

---

## Atributos

### Header

<InstanceId />
<Token />
### Path

<ParamField path="phone" type="string" required>
  Número de telefone com DDI (ex: 5511999999999)
</ParamField>
