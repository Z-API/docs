## Conceituação

Este método te permite descobrir se sua instância está ou não conectada a uma conta de WhatsApp.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

---

## Response

### 200

<ResponseField name="connected" type="boolean">
  Indica se seu número está conectado ao Z-API
</ResponseField>

<ResponseField name="error" type="string">
  Retorna detalhes do status, como "You are already connected", "You need to restore the session" ou "You are not connected"
</ResponseField>

<ResponseField name="smartphoneConnected" type="boolean">
  Indica se o celular está conectado à internet
</ResponseField>

```json
{
  "connected": true,
  "error": "You are already connected",
  "smartphoneConnected": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
