## Lista de transmissão

Uma lista de transmissão permite enviar mensagens para múltiplos contatos de uma só vez, de forma individual — cada destinatário recebe a mensagem como se fosse uma conversa direta, e eventuais respostas chegam de forma privada, sem que os outros membros da lista tenham acesso.
 
Cada lista possui um identificador com o sufixo `@broadcast`, como por exemplo: **1624901640@broadcast**.

Você pode consultar suas listas através do endpoint de [**listagem de broadcasts**](/broadcast/list-broadcast).
 
---

## Envio de mensagens para lista de transmissão

Para enviar mensagens a uma lista de transmissão, basta utilizar o endpoint de [**send-text**](/message/send-text) normalmente, passando o ID da lista de transmissão no campo `phone`.

```json
{
  "phone": "1779476888985@broadcast",
  "message": "Olá, essa é uma mensagem de broadcast!"
}
```

<Info>
O `broadcastId` pode ser obtido através do endpoint de **listagem de broadcasts** disponível na Z-API.
</Info>
