## Conceituação

Neste método você pode deletar permanentemente uma lista de transmissão existente.

---

## Atributos

### Header

<InstanceId />
<Token />

### Path

<ParamField path="broadcastId" type="string" required>
  ID da lista de broadcast a ser deletada. Ex: 1779474934@broadcast
</ParamField>

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o DELETE conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
