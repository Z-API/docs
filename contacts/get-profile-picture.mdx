---
title: "Pegar imagem do contato"
api: "GET /instances/{instanceId}/token/{token}/profile-picture"
description: "Retorna a URL com a imagem do contato atualizada"
---

## Conceituação

Este método é responsável por retornar a url com a imagem do contato atualizada.

<Info>
A imagem do contato obtida via get-contacts fica disponível por apenas **48 horas** antes do WhatsApp apagar o link. Para obter a imagem atualizada, utilize este método.
</Info>

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Query Parameters

<ParamField query="phone" type="string" required>
  Telefone do contato no formato DDI DDD NÚMERO. Ex: 551199999999
</ParamField>

---

## Response

### 200

<ResponseField name="link" type="string">
  URL com a foto do contato
</ResponseField>

```json
{
  "link": "https://..."
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
