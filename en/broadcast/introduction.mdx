## Broadcast list

A broadcast list allows you to send messages to multiple contacts at once, individually — each recipient receives the message as if it were a direct conversation, and any replies come back privately, without other list members having access.

Each list has an identifier with the `@broadcast` suffix, for example: **1624901640@broadcast**.

You can retrieve your lists through the [**list broadcasts**](/en/broadcast/list-broadcast) endpoint.

---

## Sending messages to a broadcast list

To send messages to a broadcast list, simply use the [**send-text**](/en/message/send-text) endpoint as usual, passing the broadcast list ID in the `phone` field.

```json
{
  "phone": "1779476888985@broadcast",
  "message": "Hello, this is a broadcast message!"
}
```

<Info>
The `broadcastId` can be obtained through the **list broadcasts** endpoint available in Z-API.
</Info>
