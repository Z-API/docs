---
title: "Remove participants"
api: "POST /instances/{instanceId}/token/{token}/remove-participant"
description: "Remove participants from a group"
---

## Overview

This method is responsible for removing participants from a group.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="groupId" type="string" required>
  Group ID/Phone
</ParamField>

<ParamField body="phones" type="array" required>
  Array with the phone numbers of participants to be removed
</ParamField>

---

## Request Body

```json
{
  "groupId": "120363019502650977-group",
  "phones": ["5544999999999", "5544888888888"]
}
```

---

```json
{
  "value": true
```
