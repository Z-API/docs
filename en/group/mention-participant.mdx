## Overview

This method is responsible for mentioning participants in a group.

<Tip>
You can mention multiple members at once without explicitly including "@" — just include the numbers in the `mentioned` array.
</Tip>

---

## Attributes

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Group ID where participants will be mentioned
</ParamField>

<ParamField body="message" type="string" required>
  Text to be sent (must contain @ with the number)
</ParamField>

<ParamField body="mentioned" type="array" required>
  Numbers to be mentioned
</ParamField>

### Optional

<ParamField body="delayMessage" type="number">
  Delay added to the message. You can choose between a range of 1 to 15 sec. The default delay if not provided is 1 to 3 sec
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999-group",
  "message": "Welcome to *Z-API group* @5544999999999",
  "mentioned": ["5544999999999"]
}
```
