## Overview

This method allows you to get a group's invitation link.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="groupId" type="string" required>
  Group ID/Phone
</ParamField>

---

## Response

<ResponseField name="phone" type="string">

<ResponseField name="invitationLink" type="string">
  Group invitation link
</ResponseField>
