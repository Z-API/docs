## Overview

This method returns the group metadata with all group information and its participants except for the group invitation link.

<Tip>
The only difference between this method and the Group Metadata is that this one does not return the group invitation link, as obtaining that link can be costly and slightly slow at times. If you need the link, use the [Get group invitation link](/en/group/get-invitation-link) method.
</Tip>
