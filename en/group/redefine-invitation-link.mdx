## Overview

This method allows you to reset a group's invitation link.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="groupId" type="string" required>
  Group ID/Phone
</ParamField>

---

## Response

### 200

<ResponseField name="invitationLink" type="string">
  New group invitation link
</ResponseField>

{
  "invitationLink": "https://chat.whatsapp.com/C1adgkdEGki7554BWDdMkd"
}
