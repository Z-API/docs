## Overview

This method is responsible for adding new participants to a group.

---

## Attributes

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="autoInvite" type="boolean" required>
  If unable to add directly, the API sends private invites with the group link
</ParamField>

<ParamField body="groupId" type="string" required>
  Group ID/Phone
</ParamField>

<ParamField body="phones" type="array" required>
  List of phone numbers of participants to be added
</ParamField>

---

## Request Body

```json
{
  "autoInvite": true,
  "groupId": "120363019502650977-group",
  "phones": ["5544999999999", "5544888888888"]
}
```
