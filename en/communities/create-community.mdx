## Overview

Before using this feature, it's important to verify whether the WhatsApp application on your mobile device already has community compatibility. If it's available, you can use this API to create new communities.

### 405

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
