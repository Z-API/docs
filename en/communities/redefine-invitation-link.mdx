---
title: "Redefine invitation link"
api: "POST /instances/{instanceId}/token/{token}/redefine-invitation-link/{communityId}"
description: "Redefines the invitation link of a community"
---

## Overview

This method allows you to redefine the invitation link of a community.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="communityId" type="string" required>
  Community ID
</ParamField>

---

## Response

### 200

<ResponseField name="invitationLink" type="string">
  New community invitation link
</ResponseField>

```json
{
  "invitationLink": "https://chat.whatsapp.com/C1adgkdEGki7554BWDdMkd"
}
```

### 405

Make sure you are correctly sending the method specification, that is, verify that you sent POST or GET as specified at the beginning of this topic.

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
