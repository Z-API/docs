---
title: "Link groups"
api: "POST /communities/link"
description: "Add groups to a community"
---

## Overview

With this API you can add other groups to a community.

<Warning>
It's not possible to link the same group to more than one community. If you submit 3 groups and 1 already belongs to another community, 2 will be added while the other returns an error indicating it's already part of another community.
</Warning>

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="communityId" type="string" required>
  Community ID
</ParamField>

<ParamField body="groupsPhones" type="array" required>
  Array with the IDs/Phones of the groups to be linked
</ParamField>

---

## Request Body

```json
{
  "communityId": "98372465382764532938",
  "groupsPhones": ["1345353454354354-group", "1203634230225498-group"]
}
```

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Make sure you are correctly sending the method specification, that is, verify that you sent POST or GET as specified at the beginning of this topic.

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
