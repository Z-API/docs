## Overview

This method allows you to update the community description.

<Info>
Only administrators can change community settings.
</Info>

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="communityId" type="string" required>
  Community ID
</ParamField>

<ParamField body="communityDescription" type="string" required>
  New community description
</ParamField>

---

## Request Body

```json
{
  "communityId": "120363019502650977",
  "communityDescription": "community description"
}
```
