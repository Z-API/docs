---
title: "Block / Unblock contact"
api: "POST /instances/{instanceId}/token/{token}/contacts/modify-blocked"
description: "Block or unblock a contact on WhatsApp"
---

## Overview

This method is responsible for blocking or unblocking a contact.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Contact phone number
</ParamField>

<ParamField body="action" type="string" required>
  Action to execute: `block` or `unblock`
</ParamField>

---

## Request Body

```json
{
  "phone": "5544999999999",
  "action": "block"
}
```

---

## Response

### 200

```json
{
  "value": true
}
```

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
