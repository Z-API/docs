## Overview

With this method you can vote on a specific poll.

<Info>
Voting on polls is **only available for groups**, following WhatsApp Web behavior.
</Info>

---

## Attributes

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Recipient group ID. This ID can be obtained through the get/chats method
</ParamField>

<ParamField body="pollMessageId" type="string" required>
  Poll message ID
</ParamField>

<ParamField body="pollVote" type="array" required>
  List of voted options
  <Expandable title="Properties">
    <ParamField body="name" type="string" required>
      Option name
    </ParamField>
  </Expandable>
</ParamField>
