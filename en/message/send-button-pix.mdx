## Overview

With this method you can send PIX key messages with a copy button.

<Warning>
Button sending is currently available, but there are some decisive factors for it to work. For more details check the [Button Status](/en/tips/button-status) topic.
</Warning>

<Info>
PIX messages received on WhatsApp Web don't change the chat state — they don't mark as unread or move to the top of the list. Messages render normally. This is a WhatsApp Web bug.
</Info>

---

## Attributes

### Header

<InstanceId />
<Token />
### Required

<ParamField body="phone" type="string" required>
  Recipient's phone number (or group ID for group messages) in DDI DDD NUMBER format. E.g.: 551199999999
</ParamField>

<ParamField body="pixKey" type="string" required>
  PIX key
</ParamField>

<ParamField body="type" type="string" required>
  Key type: `CPF`, `CNPJ`, `PHONE`, `EMAIL` or `EVP`
</ParamField>

### Optional

<ParamField body="merchantName" type="string">
  Button title (default: "Pix")
</ParamField>
