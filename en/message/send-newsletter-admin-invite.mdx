---
title: "Send channel admin invite"
api: "POST /instances/{instanceId}/token/{token}/send-newsletter-admin-invite"
description: "Send an invite to manage a WhatsApp channel"
---

## Overview

With this method you can send messages inviting people to become administrators of your WhatsApp channels.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Recipient's phone number in DDI DDD NUMBER format. E.g.: 551199999999. **IMPORTANT** Send only numbers, without formatting or masks
</ParamField>

<ParamField body="adminInviteMessage" type="object" required>
  Invite data
  <Expandable title="Properties">
    <ParamField body="newsletterId" type="string" required>
      Channel ID the invite belongs to. E.g.: 120363166555745933@newsletter
    </ParamField>
    <ParamField body="caption" type="string" required>
      Invite message text
    </ParamField>
  </Expandable>
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "adminInviteMessage": {
    "newsletterId": "120363166555745933@newsletter",
    "caption": "I want to invite you to be an admin of my WhatsApp channel."
  }
}
```
