---
title: "Remove reaction"
api: "POST /send-remove-reaction"
description: "Remove the reaction from a sent or received message"
---

## Overview

With this method you will remove the reaction from sent or received messages.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Recipient's phone number (or group ID for group messages) in DDI DDD NUMBER format. E.g.: 551199999999
</ParamField>

<ParamField body="messageId" type="string" required>
  ID of the message to remove the reaction from
</ParamField>

### Optional

<ParamField body="delayMessage" type="number">
  Delay added to the message. You can choose between a range of 1 to 15 sec, meaning how many seconds it will wait before sending the next message. (E.g.: "delayMessage": 5). The default delay if not provided is 1 to 3 sec
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "messageId": "3999984263738042930CD6ECDE9VDWSA"
}
```
