---
title: "Send catalog"
api: "POST /instances/{instanceId}/token/{token}/send-catalog"
description: "Send a message with the catalog link to a contact or group on WhatsApp"
---

## Overview

With this method you can send messages with a link to your catalog. The catalog feature is only available for WhatsApp **business** accounts that have registered products.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="phone" type="string" required>
  Recipient's phone number (or group ID for group messages) in DDI DDD NUMBER format. E.g.: 551199999999
</ParamField>

<ParamField body="catalogPhone" type="string" required>
  Phone number of the business account that owns the catalog
</ParamField>

### Optional

<ParamField body="translation" type="string">
  Message language (EN or PT)
</ParamField>

<ParamField body="message" type="string">
  Message sent along with the catalog card
</ParamField>

<ParamField body="title" type="string">
  Catalog card title
</ParamField>

<ParamField body="catalogDescription" type="string">
  Description in the catalog card
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "catalogPhone": "5511999999999"
}
```
