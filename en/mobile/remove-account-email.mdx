---
title: "Remove account email"
api: "POST /security/email/remove"
description: "Remove the email registered on the mobile instance's WhatsApp account"
---

## Overview

Method used to remove the email registered on your WhatsApp account.

<Info>
This API is only available for mobile instances.
</Info>

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

---

## Response

### 200

<ResponseField name="success" type="boolean">
  Returns true if the email was removed successfully
</ResponseField>

<ResponseField name="message" type="string">
  Returns `REMOVED` confirming the email removal
</ResponseField>

```json
{
  "success": true,
  "message": "REMOVED"
}
```

### 400

Invalid request. Check if the data you are sending matches the documentation.

### 405

Make sure you are correctly sending the method specification, that is, verify that you sent POST or GET as specified at the beginning of this topic.

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
