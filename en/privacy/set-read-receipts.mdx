## Overview

Through this method, you can configure message read receipts (not applicable to groups).

<Warning>
When disabling read receipts, you also won't be able to see if your own messages were read.
</Warning>

---

## Attributes

### Header

<InstanceId />
<Token />
### Query Parameters

<ParamField query="value" type="string" required>
  Setting value: `enable` or `disable`
</ParamField>
