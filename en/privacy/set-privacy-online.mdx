---
title: "Online visibility"
api: "POST /privacy/online"
description: "Configure who can see when you are online"
---

## Overview

Through this method, you can configure who can see when you are online.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="visualizationType" type="string" required>
  Visualization type: `ALL` (everyone can see) or `SAME_LAST_SEEN` (follows the same "last seen" setting)
</ParamField>

---

## Request Body

```json
{
  "visualizationType": "ALL"
}
```

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Make sure you are correctly sending the method specification, that is, verify that you sent POST or GET as specified at the beginning of this topic.

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
