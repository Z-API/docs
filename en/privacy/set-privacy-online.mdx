## Overview

Through this method, you can configure who can see when you are online.

---

## Attributes

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="visualizationType" type="string" required>
  Visualization type: `ALL` (everyone can see) or `SAME_LAST_SEEN` (follows the same "last seen" setting)
</ParamField>

---

## Request Body

```json
{
  "visualizationType": "ALL"
}
```
