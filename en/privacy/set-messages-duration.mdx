---
title: "Temporary messages"
api: "POST /instances/{instanceId}/token/{token}/privacy/messages-duration"
description: "Configure temporary messages for new individual conversations"
---

## Overview

Through this method, you can configure temporary messages for new individual conversations.

<Info>
This setting only affects **new conversations**, it is not retroactive for existing ones.
</Info>

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Query Parameters

<ParamField query="value" type="string" required>
  Message duration: `days90` (90 days), `days7` (7 days), `hours24` (24 hours) or `disable`
</ParamField>

---

```json
{
```
