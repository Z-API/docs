## Overview

With this method you can send a call to a number that has WhatsApp.

---

## Attributes

### Header

<InstanceId />
<Token />
### Required

<ParamField body="phone" type="string" required>
  Recipient's phone number in DDI DDD NUMBER format. E.g.: 551199999999
</ParamField>

### Optional

<ParamField body="callDuration" type="number">
  Call duration in seconds. Default: 5 seconds. Maximum: 15 seconds
</ParamField>

<ParamField body="callAudioUrl" type="string">
  Audio file link (16k bitrate). Only available for accounts with call functionality enabled
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "callDuration": 5
}
```
