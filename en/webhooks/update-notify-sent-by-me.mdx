## Overview

This endpoint allows you to enable the option to receive messages sent by you through the webhook.

<Info>
You must configure a webhook for the "On receive" event for this feature to work.
</Info>

---

## Attributes

### Header

<InstanceId />
<Token />
### Required

<ParamField body="notifySentByMe" type="boolean" required>
  Enable webhook for messages received and sent by me
</ParamField>

---

## Request Body

```json
{
  "notifySentByMe": true
}
```

---

```json
{
```
