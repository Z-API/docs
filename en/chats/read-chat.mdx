## Overview

This method is responsible for reading a chat as a whole, or also marking a chat as unread.

---

## Attributes

### Header

<InstanceId />
<Token />
### Required

<ParamField body="phone" type="string" required>
  Phone number of the chat you want to modify
</ParamField>

<ParamField body="action" type="string" required>
  Action: `read` or `unread`
</ParamField>

---

## Request Body

```json
{
  "phone": "5544999999999",
  "action": "read"
}
```
