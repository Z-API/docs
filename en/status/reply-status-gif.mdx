## Overview

Method responsible for sending a GIF reply to a status.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<InstanceId />
<Token />

<ParamField body="phone" type="string" required>
  Phone number of the person who posted the status
</ParamField>

<ParamField body="gif" type="string" required>
  GIF link, must be MP4
</ParamField>

<ParamField body="statusMessageId" type="string" required>
  Status message ID, obtained from the received message webhook
</ParamField>

---

## Request Body

```json
{
  "phone": "5544999999999",
  "gif": "https://example.com/file.mp4",
  "statusMessageId": "1F606398F2ECAA4846269F659B6003A9"
}
```
