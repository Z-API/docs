---
title: "Connect with phone"
api: "GET /instances/{instanceId}/token/{token}/phone-code/{phone}"
description: "Connect to the instance using a phone number instead of QRCode"
---

## Overview

Allows connection without scanning the QRCode, using the "Connect with phone number" feature.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="phone" type="string" required>
  Phone number with country code (e.g.: 5511999999999)
</ParamField>

---
