## Overview

This method allows you to check whether your instance is connected to a WhatsApp account or not.

## Response

### 200

<ResponseField name="connected" type="boolean">
  Indicates whether your number is connected to Z-API
</ResponseField>

<ResponseField name="error" type="string">
  Returns status details, such as "You are already connected", "You need to restore the session" or "You are not connected"
</ResponseField>

<ResponseField name="smartphoneConnected" type="boolean">
  Indicates whether the phone is connected to the internet
</ResponseField>

```json
{
  "connected": true,
  "error": "You are already connected",
  "smartphoneConnected": true
}
```

### 405

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
