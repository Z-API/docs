## Overview

Through this method you define the message that will be sent after rejecting a voice call received by the API.

<Warning>
For the message to be sent, the previous method ([Reject calls](/en/instance/update-call-reject-auto)) must be enabled!
</Warning>

---

### Header

<ParamField header="Client-Token" type="string">
  [Account security token](/en/security/client-token)
</ParamField>

### Body

<ParamField body="value" type="string" required>
  Response message
</ParamField>
