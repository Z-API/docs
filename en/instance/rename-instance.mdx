## Overview

Method used to rename an instance.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Body

<ParamField body="value" type="string" required>
  New name for the instance
</ParamField>

```json
{
  "value": "new name"
```
