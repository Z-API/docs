---
title: "Auto-read status"
api: "PUT /instances/{instanceId}/token/{token}/update-auto-read-status"
description: "Enable or disable automatic reading of all status posts received by the API"
---

## Overview

This method enables automatic reading of all status posts received by the API.

<Warning>
For this to work, you must have **Auto-read messages** enabled.
</Warning>

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Body

<ParamField body="value" type="boolean" required>
  `true` to enable auto-read status or `false` to disable
</ParamField>

---

## Request Body

```json
{
  "value": true
}
```

```json
{
  "value": true
```
