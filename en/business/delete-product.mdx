---
title: "Delete product"
api: "DELETE /products/{productId}"
description: "Delete a product by ID"
---

## Overview

With this method you can delete a product by its ID.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="productId" type="string" required>
  Product ID to be deleted
</ParamField>

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Make sure you are correctly sending the method specification, that is, verify that you sent DELETE as specified at the beginning of this topic.

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
