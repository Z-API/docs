title: "Delete collection"
api: "DELETE /catalogs/collection/{collectionId}"

---

## Overview

This method is responsible for deleting an existing collection from the WhatsApp Business catalog.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Path

<ParamField path="collectionId" type="string" required>
  ID of the collection to be deleted
</ParamField>

---

  "success": true
}
