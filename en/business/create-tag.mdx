---
title: "Create new tag"
api: "POST /instances/{instanceId}/token/{token}/business/create-tag"
description: "Create a new tag for chat assignment"
---

## Overview

With this method you can create a new tag available for assignment to chats.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="name" type="string" required>
  Tag name
</ParamField>

### Optional

<ParamField body="color" type="number">
  Desired color index as available in the colors API
</ParamField>

---

## Request Body

```json
{
  "name": "Tag name",
  "color": 1
}
```

---

## Response

### 200

<ResponseField name="id" type="string">
  Created tag ID
</ResponseField>

```json
{
  "id": "10"
}
```

### 405

### 415

If you receive a 415 error, make sure to add the "Content-Type" header to your request, which in most cases is "application/json".
