## Overview

Using this method you can edit your catalog settings.

---

## Attributes

### Header

<ParamField header="Client-Token" type="string">
  Account security token
</ParamField>

### Required

<ParamField body="cartEnabled" type="boolean" required>
  Enable or disable the cart
</ParamField>

---

```json
  "cartEnabled": true
}
```
