## Conceituação

Através deste método, é possível configurar as confirmações de leitura de mensagens (não aplicável a grupos).

<Warning>
Ao desabilitar as confirmações de leitura, você também não conseguirá ver se suas próprias mensagens foram lidas.
</Warning>

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Query Parameters

<ParamField query="value" type="string" required>
  Valor da configuração: `enable` (habilitar) ou `disable` (desabilitar)
</ParamField>

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
