## Conceituação

Esse é o webhook de retorno do status do chat.

<Warning>
O Z-API não aceita webhooks que não sejam **HTTPS**.
</Warning>

<Info>
Para ver todos os exemplos de retorno por status, consulte a página [Exemplos de retorno de Status do chat](/webhooks/on-chat-presence-examples).
</Info>

---

## Atualizar webhook

### Header

<InstanceId />
<Token />
### Body

<ParamField body="value" type="string" required>
  URL do webhook de presença do chat
</ParamField>

### Request Body

```json
{
  "value": "https://endereco-do-seu-sistema.com.br/instancia/SUA_INSTANCIA/presence"
}
```

---

## Response

### 200

```json
{
  "value": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou PUT conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
