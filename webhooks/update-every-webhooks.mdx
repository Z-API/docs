---
title: "Atualizar todos webhooks"
api: "PUT /instances/{instanceId}/token/{token}/update-every-webhooks"
description: "Atualiza todos os webhooks para a mesma URL de uma só vez"
---

## Conceituação

Esse endpoint serve para você que deseja alterar todos os webhooks para a mesma URL de uma só vez.

<Warning>
O Z-API não aceita webhooks que não sejam **HTTPS**.
</Warning>

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="value" type="string" required>
  URL do webhook
</ParamField>

### Opcionais

<ParamField body="notifySentByMe" type="boolean">
  Habilita notificações de webhook para mensagens recebidas e enviadas por mim
</ParamField>

---

## Request Body

```json
{
  "value": "https://endereco-do-seu-sistema.com.br/instancia/SUA_INSTANCIA/webhook",
  "notifySentByMe": true
}
```

---

## Response

### 200

```json
{
  "value": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou PUT conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
