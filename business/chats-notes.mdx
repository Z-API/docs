---
title: "Atribuir anotações a um chat"
api: "POST /instances/{instanceId}/token/{token}/chats/{phone}/notes"
description: "Atribui notas a um chat"
---

## Conceituação

Através deste método, é possível atribuir notas a um chat no WhatsApp Business.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Path

<ParamField path="phone" type="string" required>
  Telefone do chat
</ParamField>

### Obrigatórios

<ParamField body="notes" type="string" required>
  Texto da anotação
</ParamField>

---

## Request Body

```json
{
  "notes": "anotação"
}
```

---

## Response

### 200

```json
{
  "success": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
