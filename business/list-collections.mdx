---
title: "Listar coleções"
api: "GET /instances/{instanceId}/token/{token}/catalogs/collection"
description: "Lista todas as coleções do catálogo do WhatsApp Business"
---

## Conceituação

Este método é responsável por listar todas as coleções cadastradas no catálogo do WhatsApp Business.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Query Parameters

<ParamField query="nextCursor" type="string">
  Cursor para paginação dos resultados
</ParamField>

---

## Response

### 200

<ResponseField name="nextCursor" type="string">
  Cursor para a próxima página de resultados
</ResponseField>

<ResponseField name="collections" type="array">
  Lista de coleções
</ResponseField>

```json
{
  "nextCursor": "AQHRi...",
  "collections": [
    {
      "id": "1072603710847740",
      "name": "Nome da coleção",
      "status": "APPROVED"
    }
  ]
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
