---
title: "Criar coleção"
api: "POST /catalogs/collection"
description: "Cria uma nova coleção no catálogo do WhatsApp Business"
---

## Conceituação

Este método é responsável por criar uma nova coleção no catálogo do WhatsApp Business, permitindo agrupar produtos de forma organizada.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="name" type="string" required>
  Nome da coleção
</ParamField>

<ParamField body="productIds" type="array" required>
  IDs dos produtos
</ParamField>

---

## Request Body

```json
{
  "name": "Nome da coleção",
  "productIds": ["121212121212", "232323232323"]
}
```

---

## Response

### 200

<ResponseField name="collectionId" type="string">
  ID da coleção criada
</ResponseField>

```json
{
  "collectionId": "123456789123"
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
