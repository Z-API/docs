## Conceituação

Método utilizado para remover o e-mail cadastrado na sua conta do WhatsApp.

<Info>
Essa API está disponível apenas para instâncias mobile.
</Info>

---

## Atributos

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

---

## Response

### 200

<ResponseField name="success" type="boolean">
  Retorna true caso o e-mail tenha sido removido com sucesso
</ResponseField>

<ResponseField name="message" type="string">
  Retorna `REMOVED` confirmando a remoção do e-mail
</ResponseField>

```json
{
  "success": true,
  "message": "REMOVED"
}
```

### 400

Requisição inválida. Verifique se os dados que você está enviando estão de acordo com o documentado.

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
