## Conceituação

Método utilizado para verificar o e-mail cadastrado na sua conta do WhatsApp. O código de verificação é enviado para o e-mail que foi cadastrado através da API de cadastro de e-mail.

<Info>
Essa API está disponível apenas para instâncias mobile.
</Info>

---

## Atributos

### Header

<InstanceId />
<Token />
### Body

<ParamField body="verificationCode" type="string" required>
  Código de verificação enviado para o email que foi cadastrado na conta
</ParamField>

---

## Request Body

```json
{
  "verificationCode": "123456"
}
```

---

## Response

### 200

<ResponseField name="success" type="boolean">
  Retorna true caso o e-mail tenha sido verificado com sucesso
</ResponseField>

```json
{
  "success": true
}
```

### 400

Requisição inválida. Verifique se os dados que você está enviando estão de acordo com o documentado.

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
