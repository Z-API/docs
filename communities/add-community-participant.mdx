## Conceituação

Este método é responsável por adicionar novos participantes à comunidade.

<Tip>
Caso não consiga adicionar diretamente, ao ativar o `autoInvite` a API envia convites privados com o link da comunidade para os participantes que não foram adicionados.
</Tip>

---

## Atributos

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="autoInvite" type="boolean" required>
  Envia link de convite da comunidade no privado
</ParamField>

<ParamField body="communityId" type="string" required>
  ID da comunidade
</ParamField>

<ParamField body="phones" type="array" required>
  Array com os números dos participantes a serem adicionados
</ParamField>

---

## Request Body

```json
{
  "autoInvite": true,
  "communityId": "120363019502650977",
  "phones": ["5544999999999", "5544888888888"]
}
```

---

## Response

### 200

```json
{
  "value": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
