## Conceituação

Neste método você poderá fixar mensagens de uma conversa, seja chats privados ou grupos.

## Atributos

### Header

<InstanceId />
<Token />
### Obrigatórios

<ParamField body="phone" type="string" required>
  Telefone (ou ID do grupo para casos de envio para grupos) do destinatário no formato DDI DDD NÚMERO. Ex: 551199999999
</ParamField>

<ParamField body="messageId" type="string" required>
  ID da mensagem que será fixada ou desafixada
</ParamField>

<ParamField body="messageAction" type="string" required>
  Ação que será executada para a mensagem: `pin` (fixar) ou `unpin` (desafixar)
</ParamField>

<ParamField body="pinMessageDuration" type="string" required>
  Tempo em que a mensagem ficará fixada: `24_hours`, `7_days` ou `30_days`. Não tem efeito no caso de desafixar uma mensagem.
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "messageId": "77DF5293EBC176FFA6A88838E7A6AD83",
  "messageAction": "pin",
  "pinMessageDuration": "7_days"
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
