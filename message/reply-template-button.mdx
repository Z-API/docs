## Conceituação

Este endpoint permite responder um botão de ação, simulando o clique do usuário em uma das opções enviadas anteriormente via [send-button-actions](/message/send-button-actions).

<Warning>
Apenas botões do tipo **REPLY** podem ser respondidos. Botões de outros tipos não suportam resposta via API.
</Warning>

<Info>
O botão deve ter sido previamente enviado e o **ID do botão** deve ser obtido via webhook de recebimento da mensagem.
</Info>

### Obrigatórios

<ParamField body="phone" type="string" required>
  Telefone (ou ID do grupo) no formato DDI DDD NÚMERO. Ex: 551199999999
</ParamField>

<ParamField body="responseTemplateButtonId" type="string" required>
  ID do botão do tipo REPLY. Esse ID é obtido via webhook da mensagem recebida.
</ParamField>

<ParamField body="messageId" type="string" required>
  ID da mensagem que contém os botões enviados anteriormente.
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999",
  "responseTemplateButtonId": "1",
  "messageId": "D241XXXX732339502B68"
}
```
