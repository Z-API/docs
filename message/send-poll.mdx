## Conceituação

Neste método você poderá enviar mensagens do tipo enquete.

<Info>
O envio de enquetes está disponível **apenas para grupos**, seguindo o comportamento do WhatsApp Web.
</Info>

## Atributos

### Header

<InstanceId />
<Token />
### Obrigatórios

<ParamField body="phone" type="string" required>
  ID do grupo destinatário. Esse ID pode ser obtido através do método get/chats
</ParamField>

<ParamField body="message" type="string" required>
  Texto a ser enviado na enquete
</ParamField>

<ParamField body="poll" type="array" required>
  Lista de opções da enquete
  <Expandable title="Propriedades">
    <ParamField body="name" type="string" required>
      Nome da opção
    </ParamField>
  </Expandable>
</ParamField>

### Opcionais

<ParamField body="delayMessage" type="number">
  Delay adicionado na mensagem. Você pode decidir entre um range de 1 a 15 sec, significa quantos segundos ele vai esperar para enviar a próxima mensagem. (Ex.: "delayMessage": 5). O delay default caso não seja informado é de 1 a 3 sec
</ParamField>

<ParamField body="pollMaxOptions" type="number">
  Número máximo de votos por pessoa na enquete
</ParamField>

## Request Body

**Múltiplas escolhas:**

```json
{
  "phone": "5511999999999-1234567890@g.us",
  "message": "Qual a melhor API para WhatsApp?",
  "poll": [
    { "name": "Z-API" },
    { "name": "Outras" }
  ]
}
```

**Escolha única:**

```json
{
  "phone": "5511999999999-1234567890@g.us",
  "message": "Qual a melhor API para WhatsApp?",
  "pollMaxOptions": 1,
  "poll": [
    { "name": "Z-API" },
    { "name": "Outras" }
  ]
}
```

---

## Response

### 200

```json
{
  "zaapId": "3999984263738042930CD6ECDE9VDWSA",
  "messageId": "D241XXXX732339502B68",
  "id": "D241XXXX732339502B68"
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
