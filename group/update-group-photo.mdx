## Conceituação

Este método é responsável por alterar a imagem de um grupo já existente.

---

## Atributos

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="groupId" type="string" required>
  ID/Phone do grupo
</ParamField>

<ParamField body="groupPhoto" type="string" required>
  URL ou Base64 da imagem
</ParamField>

---

## Request Body

```json
{
  "groupId": "120363019502650977-group",
  "groupPhoto": "https://www.z-api.io/wp-content/themes/z-api/dist/images/logo.svg"
}
```

## Response

### 200

```json
{

}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
