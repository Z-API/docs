---
title: "Adicionar participantes"
api: "POST /instances/{instanceId}/token/{token}/add-participant"
description: "Adiciona novos participantes ao grupo"
---

## Conceituação

Este método é responsável por adicionar novos participantes ao grupo.

---

## Atributos

### Header

<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="autoInvite" type="boolean" required>
  Caso não consiga adicionar, a API envia convites privados com o link do grupo
</ParamField>

<ParamField body="groupId" type="string" required>
  ID/Phone do grupo
</ParamField>

<ParamField body="phones" type="array" required>
  Lista de números de telefone dos participantes a serem adicionados
</ParamField>

---

## Request Body

```json
{
  "autoInvite": true,
  "groupId": "120363019502650977-group",
  "phones": ["5544999999999", "5544888888888"]
}
```

---

## Response

### 200

```json
{
  "value": true
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
