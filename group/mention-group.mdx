## Conceituação

Este método é responsável por fazer a menção de grupos relacionados a uma comunidade. O sistema restringe menções a grupos dentro da mesma estrutura de comunidade.

---

## Atributos

### Header

<InstanceId />
<Token />


<ParamField header="Client-Token" type="string">
  Token de segurança da conta
</ParamField>

### Obrigatórios

<ParamField body="phone" type="string" required>
  ID do grupo onde a menção será enviada
</ParamField>

<ParamField body="message" type="string" required>
  Texto a ser enviado (deve conter @ com o ID do grupo)
</ParamField>

<ParamField body="groupMentioned" type="array" required>
  Array com os dados dos grupos a serem mencionados
  <Expandable title="Propriedades">
    <ParamField body="phone" type="string" required>
      ID do grupo sendo mencionado
    </ParamField>
    <ParamField body="subject" type="string" required>
      Nome do grupo
    </ParamField>
  </Expandable>
</ParamField>

### Opcionais

<ParamField body="delayMessage" type="number">
  Delay adicionado na mensagem. Você pode decidir entre um range de 1 a 15 sec. O delay default caso não seja informado é de 1 a 3 sec
</ParamField>

---

## Request Body

```json
{
  "phone": "5511999999999-group",
  "message": "Welcome to *Z-API group* @1203634230225498-group",
  "groupMentioned": [
    {
      "phone": "1203634230225498-group",
      "subject": "Z-API subgroup"
    }
  ]
}
```

### 405

Neste caso certifique que esteja enviando corretamente a especificação do método, ou seja verifique se você enviou o POST ou GET conforme especificado no início deste tópico.

### 415

Caso você receba um erro 415, certifique de adicionar na headers da requisição o "Content-Type" do objeto que você está enviando, em sua grande maioria "application/json".
